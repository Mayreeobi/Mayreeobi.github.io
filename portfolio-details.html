Data Science Salary Analysis

Project information

  • Tools: SQL for manipulation and Power BI for visualization
  • Key Concepts: Aggregate Functions, Window Functions
  • Project URL: www.github.com
  • Interactive Dashboard: www.powerbi.com

Data Science is a rapidly growing field with diverse job opportunities, ranging from Data Engineers to Research Analysts.
This project undertakes a comprehensive qualitative analysis of data science salaries in 2023, with the primary objective of shedding light on prevailing salary trends and employment dynamics. Furthermore, the project also aims to delve deeper into the reasons behind the substantial shift of professionals towards the field of data science.
